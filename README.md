## Version Info

- NodeJS v12.16.3

## Getting Started

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

> Install Dependencies

```
$ npm install
```

> Run application

```
# For production mode
$ npm run start

# For development mode
$ npm run dev
```
